title: Kbd

---

Kbd renders keyboard keys with a realistic keyboard shortcut appearance. Use KbdGroup to display key combinations.

## Installation

<CodeGroup>
```bash npm
npm install @soft-ui/react
```
```bash yarn
yarn add @soft-ui/react
```
```bash pnpm
pnpm add @soft-ui/react
```
</CodeGroup>

## Usage

```tsx
import { Kbd, KbdGroup } from "@soft-ui/react/kbd"

<Kbd>⌘</Kbd>
<Kbd>K</Kbd>

// Or as a group
<KbdGroup keys={["⌘", "K"]} />
```

## API Reference

### Kbd

<ParamField path="variant" type="'tertiary' | 'inverted'" default="'tertiary'">
  Visual style:
  - `tertiary`: Light background (default)
  - `inverted`: Dark background with white text
</ParamField>

<ParamField path="className" type="string">
  Additional CSS classes
</ParamField>

### KbdGroup

Renders multiple keyboard keys with proper spacing.

<ParamField path="keys" type="string[]" required>
  Array of key labels to display
</ParamField>

<ParamField path="variant" type="'tertiary' | 'inverted'" default="'tertiary'">
  Visual style (applied to all keys)
</ParamField>

<ParamField path="className" type="string">
  Additional CSS classes for the container
</ParamField>

## Examples

```tsx

<Kbd>Enter</Kbd>

```

### Key Combinations

```tsx
<div className="flex items-center gap-2">
  <Kbd>⌘</Kbd>
  <Kbd>K</Kbd>

```

### Using KbdGroup

```tsx
<KbdGroup keys={["⌘", "K"]} />
```
